# CPP_Project_airplane

Modélisation simplifiée du contrôle de la circulation aérienne réalisée avec le langage C++ ainsi que la librairie SFML.

## Fichiers requis

Ce projet requiert l'utilisation du dossier __SFML_3.0.2__, contenant les différents sprites utilisés pour la simulation.

## Emplacement

Le dossier SFML_3.0.2 (__précisément__ celui présent dans ce dépôt et contenant le dossier img) nécessite d'être placé à la __racine__ de votre disque principal (C:\), ou de modifier la ligne 14 du fichier main.cpp
pour remplacer le chemin par votre emplacement du dossier.

```bash
#define _PATH_IMG_ "C:/SFML_3.0.2/img/"
```

## Version de CMake

Ce projet utilise la version __3.31__ de CMake.

**NOTE:** Il est possible que cette version de CMake, présente dans le fichier CMakeLists.txt placé à la racine du projet, ne convienne 
pas à certaines machines lors de la compilation. Il est alors nécessaire de passer à une version ultérieure.
